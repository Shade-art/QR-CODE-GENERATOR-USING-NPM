# 📌 QR Code Generator

A simple Node.js CLI app that generates a QR code from a URL entered by the user.

---

## 🚀 Features

* Interactive input using **Inquirer**
* Generates QR code instantly
* Saves output as `qr_img.png`

---

## 🛠️ Tech Stack

* Node.js
* Inquirer
* qr-image
* fs

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/qr-code-generator.git
cd qr-code-generator
npm install
```

---

## ▶️ Usage

```bash
node index.js
```

Enter a URL → QR code will be saved as `qr_img.png`.
